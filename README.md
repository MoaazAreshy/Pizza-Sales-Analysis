
# 🍕 Pizza Sales & Customer Preferences Analysis

## Overview

This project focuses on analyzing **pizza restaurant sales and customer preferences** using a complete data analytics workflow.
The goal is to extract meaningful insights from the dataset and present them through **SQL analysis and an interactive Power BI dashboard** to support data-driven decision making.

The project follows a typical **Data Analytics pipeline**, starting from data exploration to building a final dashboard and presentation.

---

## Dataset

The dataset contains information about pizza orders, including:

* Order details
* Pizza categories and sizes
* Quantity sold
* Order dates and times
* Sales and revenue information

This dataset was used to analyze **sales trends, popular pizza types, and customer preferences**.

---

## Tools & Technologies

The following tools were used throughout the project:

* **SQL Server** → Data exploration, cleaning, and analytical queries
* **Power BI** → Data modeling and building an interactive dashboard
---

## Project Steps

### 1. Data Exploration

The dataset was explored using **SQL queries** to understand its structure, check for missing values, and examine key columns.

### 2. Data Cleaning

Data preparation included:

* Handling missing or inconsistent values
* Adjusting data types
* Preparing the dataset for analysis

### 3. SQL Analysis

Several analytical queries were written in **SQL Server** to extract insights such as:

* Total revenue
* Best-selling pizzas
* Sales by category and size
* Order trends over time

### 4. Data Modeling

The dataset was prepared for visualization in **Power BI**, applying proper relationships and modeling to support analysis.

### 5. Power BI Dashboard

An interactive **Power BI dashboard** was created to visualize the insights including:

* Total sales and revenue
* Top-selling pizza types
* Sales by category and size
* Order trends over time

---

## Dashboard

The Power BI dashboard provides a clear overview of:

* Revenue performance
* Most popular pizzas
* Customer ordering trends
* Sales distribution by category and size

Interactive visuals allow users to quickly explore insights and identify key patterns in sales data.

---

## Results & Insights

Key insights discovered during the analysis include:

* Some pizza categories generate the highest revenue.
* A small number of pizza types account for a large portion of sales.
* Sales patterns vary depending on time and demand.
* Understanding customer preferences can help improve menu strategy and sales performance.
